The Web

The web is a collection of documents written in HTML along with other resources such as images and videos, as well as programs and services that interact with the documents and resources.
Hypertext:a form of text in which documents can refer(link) to other documents and resources. The web is a hypertext system.

The browser gets web pages by communicating with servers over the internet.
Servers are computers that have programs running on them which answer a browser's request

Hypertext Transfer Protocol (HTTP): the rules for how the requests and responses between the browser and server work. HTTP= insecure and HTTPS= secure.

HTML

HyperText Markup Language(HTML):The language that provides the structure and text of web pages.

Things To Remember:

Syntax:Grammar rules of a language.
Formalism:Computers take code literally, and will only do exactly what you tell it to do.
Nesting:Bits of code going inside other bits of code in an orderly way. Elements have to go completely inside each other. They can not overlap.

HTML is written in text. The layout of a webpage seen by the user is put together by the web browser based on the code in the text file.

Text files contain:
  • Letters
  • Numbers
  • Symbols
  • Spaces

Text editors are used to create and edit text files. Each text editor has it's own features like syntax highlighting and line numbers.

Save HTML files with .html at the end of the file name.
In HTML there is text that the user will actually read in their browser. Markup:text that has a special meaning, tells the browser what that text should look like or how it's arranged. And within markup, there can be... References to include other documents and files, like images and videos. Plain text in HTML shows up as plain text in the browser.

Elements and Tags


An element has an open and closed angle bracket with a keyword in between on both sides of a piece of text.
Opening tag has an open and closed angle bracket with a keyword in between and closing tag is like an opening one but with a / after the open bracket.
All whitespace or spaces, tabs and line breaks in this file was made using the p or br tag.
P was used to separate paragraphs and br was used to end a line.
Div tags are used to divide the page into different sections. It was used for the areas where there is a list.
